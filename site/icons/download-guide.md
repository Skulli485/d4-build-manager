# Klassen-Icons herunterladen und installieren

## Option 1: Von vitablo.de extrahieren

1. Besuche [vitablo.de](https://vitablo.de/diablo-4-build/)
2. Öffne die Entwickler-Tools (F12)
3. Lade die Icons über das Netzwerk-Tab herunter
4. Speichere sie im `icons/` Ordner

## Option 2: Offizielle D4 Icons verwenden

Die Icons sind im `icons/` Ordner zu platzieren:

```bash
icons/
├── barbar.png        # Barbar
├── druid.png         # Druide
├── necromancer.png   # Totenbeschwörer
├── paladin.png       # Paladin
├── rogue.png         # Jägerin
├── sorceress.png     # Zauberin
├── spiritborn.png    # Spiritborn
└── warlock.png       # Hexenmeister
```

## Empfohlene Icons-Quellen

- **Vitablo.de**: Die verwendet die gleichen Icons wie auf der Website
- **Diablo4.com**: Offizielle Blizzard-Assets
- **Diablo Wiki**: Community-Ressourcen

## Icon-Format

- **Größe**: 64x64 oder 128x128 Pixel
- **Format**: PNG mit Transparenz
- **Stil**: Offizielle D4 Klassen-Symbole

## Fallback

Solange die Icons nicht vorhanden sind, werden automatisch Emojis angezeigt:

- Paladin: ⚔️
