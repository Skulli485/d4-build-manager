# Diablo 4 Klassen-Icons

Platziere die offiziellen D4 Klassen-Icons in diesem Ordner.

## Benötigte Icons:

| Klasse | Dateiname | Emoji Fallback |
|--------|-----------|----------------|
| Barbar (Barbar) | `barbar.png` | ⚔️ |
| Druid (Druide) | `druid.png` | 🐻 |
| Necromancer (Totenbeschwörer) | `necromancer.png` | 💀 |
| Paladin | `paladin.png` | ⚔️ |
| Rogue (Jägerin) | `rogue.png` | 🗡️ |
| Sorceress (Zauberin) | `sorceress.png` | ✨ |
| Spiritborn (Geistgeborener) | `spiritborn.png` | 🦅 |
| Warlock (Hexenmeister) | `warlock.png` | 🔮 |

## Icon-Quellen:

Offizielle Icons können von folgenden Seiten bezogen werden:
- [Diablo4.com](https://diablo4.blizzard.com/) - Offizielle Website
- [Vitablo.de](https://vitablo.de) - Extrahiert aus den Build-Seiten
- [Diablo Wiki](https://diablo.fandom.com/wiki/Diablo_IV)

## Empfohlene Größe:
- 64x64 Pixel oder größer
- PNG mit Transparenz
- Quadratisch

## Hinweis:
Die Icons werden automatisch geladen, wenn sie im `icons/` Ordner vorhanden sind.
Solange keine Icons vorhanden sind, werden die Emojis als Fallback verwendet.
