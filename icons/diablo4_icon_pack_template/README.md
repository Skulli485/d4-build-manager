# Diablo 4 Icon Pack Template — Legal Placeholder Edition

Dieses Paket enthält **keine offiziellen Blizzard-/Diablo-IV-Icons**. Es enthält selbst erstellte Platzhalter-SVGs und eine saubere Manifest-Datei, damit du ein Icon-Pack strukturieren kannst, ohne urheberrechtlich geschützte Originalgrafiken weiterzuverteilen.

## Inhalt

- `icons/` — Ordner nach Klasse und Skill-Kategorie
- `manifest.json` — Liste aller enthaltenen Einträge mit Klasse, Kategorie, Skillname und Dateipfad

## Umfang

Enthalten sind die Skill-Tree-Einträge für:

- Barbarian
- Rogue
- Sorcerer
- Necromancer
- Druid
- Spiritborn

Anzahl der Einträge: 384

## Hinweis

Wenn du offizielle Icons privat für dein eigenes Interface verwenden willst, solltest du sie nur aus rechtmäßig zugänglichen Quellen beziehen und nicht weiterverteilen. Für öffentliche Projekte, Webseiten, Apps oder Downloads besser eigene Icons nutzen oder eine ausdrückliche Lizenz einholen.
